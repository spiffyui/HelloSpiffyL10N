Il s'agit d'un Spiffy interface utilisateur application construite pour vous et en cours d'exécution sur votre ordinateur. Ce simple exemple montre que vous sous une forme aisément ce qui en fait un REST appel à votre serveur. Vous pouvez l'éditer, le construire, l'exécuter, de le partager avec vos amis, et créer un produit au-dessus de celui-ci. La demande est tout à vous.

Prochaines étapes

Vous avez joué avec le projet un peu, maintenant faire quelques changements.

  1. Changer cette page - Cette page est un modèle avec la plupart du code HTML et les contrôles dynamiques ajouté plus tard. Modifier le contenu dans src/main/java/org/spiffyui/hellospiffylocalization/client/MainPanel.html.
  2. Changer le style - Vous pouvez définir n'importe quel style dans src/main/java/org/spiffyui/hellospiffylocalization/public/HelloSpiffyLocalization.css.
  3. Ajouter un REST Bean - Faire de cette REST appel réutilisables en l'ajoutant à une REST Bean.
  4. Ajouter un bar navigation - Une navigation facilitée pour votre application est l'une des grandes caractéristiques de Spiffy assurance-chômage.

Après avoir apporté tout changement, vous pouvez arrêter le serveur et exécuter le Ant commande pour recompiler votre projet. Vous pouvez également utiliser le Eclipse IDE pour exécuter cette application dans le GWT mode dev.
